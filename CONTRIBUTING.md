# Contributing

Contributions are welcome! Whether it's a bug fix, or new feature we'd love your help.

## Making changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Run `cargo build` and test on your setup
4. Open a pull request

## Guidelines

- Keep PRs focused on a single change
- Follow the existing code style
- No unnecessary dependencies

## Reporting bugs

Open an issue with:

- Your platform and compositor/window manager version
- Steps to reproduce
- Expected vs actual behavior
- A short video if possible
